What is Polite?

Polite enables you to automatically silence your phone at the right times. To do this, you need to create rules so that Polite knows when to silence your phone. There are currently two types of rules: Schedule Rules and Calendar Rules. For every rule, you can choose to enable vibrate mode instead of silent mode. If multiple rules apply at the same time, Polite will give precedence to rules without vibrate.

Schedule Rules

Schedule Rules allow you to silence your phone at a specific time on specific days of the week. This works well if you have a weekly event or work schedule where you want your phone to be in vibrate mode or silent mode.

If you select an end time that is before the start time, then the end time will be on the following day. For instance, if you make a rule for Friday 10pm - 6am, then Polite will activate on 10pm on Friday and deactivate on Saturday at 6am.

Calendar Rules

Calendar Rules allow you to silence your phone during calendar events saved in your calendar app.

Calendars

Select specific calendars to look for events or just use "All" calendars.

Events

Select the method used to match calendar events. You can simply match "All Events" or match events by keyword in the event title, description or both.

Keywords

When using keywords, Polite will activate for events that contain any one of the keywords in a rule. Keywords are not case-sensitive. A keyword can have multiple words. Partial word matches do count.

Examples:

  • "dinner" matches "Meet Suzie for dinner"
  • "tray" matches "Betrayal On Elm Street"
  • "meeting" matches "MANAGER MEETING"
  • "mat 100" matches "MAT 100 - Adding and Subtracting"
  • "bob lunch" does not match "Meet Bob for lunch"

Inverse Match

By enabling Inverse Match, Polite will activate for rules that DO NOT contain keywords. The rule still only applies to events in the calendars you have selected.

Settings

Notifications

When enabled, you will see a notification whenever Polite is active.

Activation

Activate Polite a specified amount of time before an event starts

Deactivation

Deactivate Polite a specified amount of time after an event ends

Priority Mode

If you have Android 5.0 or higher, your phone will use Priority Mode instead of Silent Mode. This does not apply to rules where Vibrate is enabled. In Priority Mode, "priority interruptions" will still be able to ring or vibrate. You can change what qualifies as a priority interruption by going to Settings -> Sound & notification -> Interruptions.

Tips

You can use a unique tag (such as #vibrate) as a keyword and then put that tag in your event descriptions where you want to apply the rule.

You can create a separate calendar for your Polite schedule and create a rule to match all events in that calendar. Then you can hide the calendar in your calendar app if you want to avoid clutter.
